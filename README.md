Repository ini berisi kumpulan latihan belajar Python dasar untuk pemula menggunakan Visual Studio Code dengan terminal bawaan Windows.

## Tujuan
- Memahami dasar bahasa pemrograman Python
- Melatih logika pemrograman
- Membiasakan penggunaan VS Code dan terminal

## Materi
- Pengenalan Python
- Input dan Output
- Variabel dan Tipe Data
- Percabangan (if / else)
- Perulangan (for / while)
- Fungsi sederhana

## Tools
- Python 3
- Visual Studio Code
- Windows Terminal (bawaan VS Code)

## Cara Menjalankan Program
1. Buka folder project di VS Code
2. Buka terminal (Ctrl + `)
3. Jalankan perintah berikut:
```bash
python nama_file.py
